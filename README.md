# Privacy Bulk Actions for Caflou

[Čeština](#čeština) | [English](#english)

---

## Čeština

### O rozšíření
**Privacy Bulk Actions for Caflou** je rozšíření pro prohlížeč Google Chrome, které přidává hromadné akce pro správu přístupových práv a soukromí v obchodních případech a projektech v systému Caflou.

### Účel
Systém Caflou standardně vyžaduje manuální nastavování práv (čtení, zápis, finance/zdraví) pro každého uživatele zvlášť. Toto rozšíření přidává do uživatelského rozhraní nástrojovou lištu, která umožňuje jedním kliknutím:
- **Vše odškrtnout**
- **Vše obnovit (dle role)**
- **Všem povolit vše**

Výrazně tím urychluje práci při nastavování složitějších oprávnění.

### Instalace

**Z Google Chrome Web Store (Doporučeno)**
1. Přejděte na stránku rozšíření v [Google Chrome Web Store](https://chrome.google.com/webstore) *(odkaz bude doplněn po publikaci)*.
2. Klikněte na tlačítko **Přidat do Chromu**.
3. Rozšíření bude automaticky nainstalováno a připraveno k použití.

**Lokální instalace (pro vývojáře)**
1. Stáhněte si tento repozitář nebo jej naklonujte pomocí `git clone`.
2. Otevřete prohlížeč Chrome a přejděte na adresu `chrome://extensions/`.
3. V pravém horním rohu zapněte **Režim pro vývojáře**.
4. Klikněte na **Načíst rozbalené** (Load unpacked) a vyberte složku s tímto repozitářem.

## English

### About the Extension
**Privacy Bulk Actions for Caflou** is a Google Chrome extension that adds bulk actions for managing access rights and privacy settings in the Caflou system.

### Purpose
The Caflou system typically requires setting rights (read, write, health/finance) manually for each user individually. This extension adds a toolbar to the user interface that allows you to do the following with a single click:
- **Clear all**
- **Restore all (by role)**
- **Grant all to everyone**

This significantly speeds up the workflow when configuring complex business cases and projects.

### Installation

**From Google Chrome Web Store (Recommended)**
1. Go to the extension's page on the [Google Chrome Web Store](https://chrome.google.com/webstore) *(link to be added after publication)*.
2. Click the **Add to Chrome** button.
3. The extension will be automatically installed and ready to use.

**Local Installation (for developers)**
1. Download this repository or clone it using `git clone`.
2. Open Chrome and navigate to `chrome://extensions/`.
3. In the top right corner, enable **Developer mode**.
4. Click **Load unpacked** and select the folder containing this repository.
